Verkefni 1

Vefsíðugerð

Þegar hafist var á verkefninu var fyrst farið inn á síðuna HTML5 UP til að finna flott vefsíðu sniðmát til að vinna með. Helios síðan var valin sem grunnur og unnið út frá því. En erfiði parturinn var að vita hvar ætti að byrja. Þá var þægilegt að styðjast við myndböndin hjá kennaranum sem eru eftirfarandi.

  • Myndskeið frá kennara

Til þess að gera vefsíðuna þá hlóð ég niður Brackets. Þegar það var komið færði ég möppuna fyrir vefsíðu sniðmátið yfir í Brackets og byrjaði að fikta mig áfram. Ég komst að því fljótt að vefvafrinn minn (Brave browser) virkaði ekki til að sjá vefsíðuna samhliða þegar maður er að kóða. Þá þurfti ég að hlaða niður chrome aftur sem var ekki ákjósanlegt þar sem Chrome fer ekki vel með Mac-a. En þegar vefsíðan virkaði live og ég gat farið að kóða þá studdi ég mig mikið við w3schools fyrir grunnþekkingu.

Til að gera vefsíðuna eins og mig langaði að hafa hana, eyddi ég út óþarfa flipumm, textum og myndum. Notaði tilbúin sniðmót um left sidebar og no sidebar eftir því hvernig ég vildi að síðan væri uppsett. Svo notaðist ég mikið við ChatGPT til þess að hjálpa mér að setja ferilskrána sem pdf, breyta litum og breyta buttons svo þeir virki til að hlaða niður ferilskránni. Einnig hjálpaði chat mér við að tengja LinkedIn og tölvupóstinn minn neðst á síðunni. Svo var farið í allskonar dundur við að breyta litum og setja inn myndir. Til að breyta myndum eins og ég vildi hafa þær notaði ég til dæmis eftirfarandi skipun, þessi skipun var notuð fyrir myndina á Um mig síðunni.

Hvað vil ég fá út úr áfanganum

Ég er mjög spennt fyrir þessum áfanga þar sem það er mikið frelsi við gerð verkefnanna og mikil sköpun á bakvið þau. Ég vil öðlast meiri þekkingu á 3D-prentun og kunna ferlið á bakvið það. Svo vil ég vera öruggari í því að nota CAD-teikningar í hluti sem raunverulega er hægt að notfæra sér. Ég hef ekki mikið pælt í því hvað ég ætla að gera sem lokaverkefni en ég vil gera eitthvað sem ég get notfært mér og endar ekki í geymslunni. Það er eitthvað í herbergið mitt sem væri mér til taks dags daglega.

Uppsetning á Git og Github

Til þess að setja upp Git og Github fór ég mikið eftir leiðbeiningunum hjá Björg Þar sem þær voru fyrir mac sem var einstaklega þægilegt. Ég hlóð niður Homebrew með þessari skipun í terminal. 

                                    
                                        /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Næst var þessi skipun sett í terminal: 

                                    
                                         $ brew install git

Næst var repository á Github stofnað. Búinn var til aðgangur á Github og næst var ýtt á create repository

Svo var farið í að fylla út í upplýsingar um repositoryið, það þarf að vera public og valið var nafn á það og klikkað á create repository .

Svo var farið í möppuna í tölvunni þar sem skrárnar eru geymdar og hægri klikkað og ýtt á New Terminal at folder Þá poppaði upp terminalið.

Svo að lokum til að hlaða skránum inn á Github síðuna voru eftirfarandi comment sett í terminalið: 

                                    
                                        echo "# helgarun" >> README.md
                                        git init
                                        git add .
                                        git commit -m "first commit"
                                        git branch -M main
                                        git remote add origin https://github.com/helgahermanns/helgarun.git
                                        git push -u origin main

Vesen var að skrá mig inn á githubið í terminalinu með lykilorðinu því bjó ég til Token eftir þessari síðu sem svínvirkaði og allar skrárnar voru komnar inn.

Endurhlaða breytingum á Github

Þegar ég breytti einhverju á síðunni þurfti ég að fara aftur í gegnum skrefin með því að hlaða möppunum inn á Github. En þá var eftirfarandi slegið inn í terminalið: 

                                    
                                        git add .
                                        git commit -m "eitthvað comment"
                                        git push

Einnig er hægt að draga inn skrárnar ef þeim var breytt og þá uppfærast skrárnar, ég notaðist við báðar aðferðir.
